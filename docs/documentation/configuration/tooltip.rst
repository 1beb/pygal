Tooltip
=======

Tooltips are displayed when the pygal javascript is used.


tooltip_border_radius
---------------------

.. pygal-code::

  chart = pygal.Line(tooltip_border_radius=10)
  chart.add('line', [.0002, .0005, .00035])


tooltip_font_size
-----------------

.. pygal-code::

  chart = pygal.Line(tooltip_font_size=24)
  chart.add('line', [.0002, .0005, .00035])
